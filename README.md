# Python C2 Architecture Lab

This project is an educational command-and-control (C2) architecture built in Python ONLY for learning purposes.

Features:
- Encrypted payload transport (Fernet)
- Persistent agent identity
- Agent heartbeat + status tracking
- Operator CLI
- Modular project structure

This project is intended strictly for educational and research use to further my understanding of cybersecurity concepts.

Credit goes to: https://medium.com/maxwell-cross-python-for-red-teaming/building-a-custom-c2-server-in-python-a-fresh-take-on-offensive-security-e3d8c09bc2ab for the idea and basic architecture!
